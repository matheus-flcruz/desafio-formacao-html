Estrutura Básica

Voltar

Basicamente o HTML e formado pelas tags Head e pelo Body.

Head e tudo que a página precisa carregar antes de abrir o site, como por exemplo o título que fica na aba do navegador. Já o Body e a página em si, tudo o que as pessoas vem no navegador, por exemplo textos, imagens, videos e tudo o que aparece na página.

Tags

Voltar

As Tags são como marcadores que são usados para estruturar a página, definindo o que e título, subtítulo, paragráfos, imagens e etc.

Eles devem sempre estar entre os sinas de menor que (<) que é a tag de abertura e a maior que (>) que é a tag fechamento. Com exceção de alguns tags, todas elas tem que abrir e fechar. A tag de fechamento sempre terá o caracter (/) antes do maior que.

Exemplo de algumas tags mais usadas são: (h) de header, que é para títulos, cabeçalhos ou subtítulos e tem níveis que vão do h1 até o h6. A tag (p) de paragraph, que como o nome diz, e para começar parágrafos. A tag (i) de itálico, para deixar alguma palavra ou o texto em itálico. A tag (strong) para deixar alguma palavra ou texto em negrito.

Já exemplos de tags que não tem fechamento, temos a (input) que abre uma caixa para digitação no sites, como aquelas de login e senha e a tag img que é uma tag para inserir imagens do site.

Atributos

Voltar

Atributos são informações adicionais para as tags, como definir uma função, cor e tamanho da fonte, entre outras propriedades. Existem os atributos Globais, que são mais genêricos e servem para vários tipos de tags e os atributos Especifícos, que como o nome já diz, são especifícos para cada tag.

Exemplos de atributos globais são: (id) que serve como um identificador da tag, a (style)que serve para você um comando na tag e (class) que serve criar uma classe para as tags. Já como exemplo de atributos especifícos temos a tag (type) que é um atributo especifíco da tag (input).

Textos

Voltar

Basicamente são como os textos do site serão configurados e estilizados. Definindo o que são títulos, subtítulos usando a tag (h), paragráfos usando a tag (p), textos em formato de citacões usando a tag (blockquote), para deixar o texto pequeno usando a tag (small).

Listas Ordenadas e Não Ordenadas

Voltar

Listas são indíces ou tópicos. As principais são a Ordenada e Não Ordenada.

Para criar essas listas são usadas as tags (ol) para a Lista Ordenada e (ul) para Lista Não Ordenada.

A diferença entre as duas e que a Ordenada é classificada por ordem númerica (1., 2., 3...) enquanto a Não-Ordenada é apenas uma lista de itens (•,-,■).

Mas essas tags sozinhas não criam as listas. Para que seja possível digitar textos e de fato criar as listas e usada a tag (li) que são os Item da Lista. Ela é uma tag criada dentro das tags (ol) ou (ul) e na (li) e colocado o texto do indíce.

Links

Voltar

Nos sites existem links que levam para outros sites ou para algum ponto especifíco da página.

Para criar esses links usamos a tag (a) e usamos o atributo (target) para definirmos se a próxima página será aberta em outra janela ou na mesma janela.
